# Counter Application 🔢

A simple interactive counter app built using HTML, CSS, and JavaScript.
This project demonstrates basic DOM manipulation and event handling.

## 🚀 Tech Stack

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## ✨ Features

* Increase counter by 1
* Decrease counter by 1
* Increase counter by 10
* Reset counter to 0
* Dynamic background color changes based on action

## 📚 What I Practiced

* DOM manipulation using `getElementById`
* Handling events with `addEventListener`
* Updating UI dynamically using JavaScript
* Working with `textContent` and type conversion (`parseInt`)

## 📌 Future Improvements

* Add minimum and maximum limits
* Add animations and transitions
* Improve UI design with better styling
* Add sound or visual feedback for actions

## 📁 Project Status

Completed as a revision project to strengthen JavaScript fundamentals and interactivity.
